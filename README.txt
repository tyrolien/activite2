Le projet consiste en un exercice qui a pour objectif de mettre en pratique les notions vues dans les deux premières parties de ce cours sur Git et Github.

Pour cela, il faut créer un repository Git simple contenant :

    Un fichier nommé "README.txt" qui présente le projet;
    Au moins 2 autres fichiers de votre choix (ils peuvent contenir des scripts de code, du texte sur un projet personnel...);
    Au moins 4 commits distincts montrant l’ajout puis la modification de certains fichiers.

On va simuler la création et modification d'une forme HTLM d'inscription avec son code css.
